# WorkoutHelperJW
Workout Helper – by Jacob Wu

Install all programs and run the WHMain.py to get started.

This program lets you track your workouts and see your progress. It tracks time, distance/kj, and even laps. You can also view previous workouts and its laps. Can't find the workout you want? Use the "Contact me" page to suggest new workouts!
